# 医疗护理指南

## 伤势分级处理
- **轻微擦伤/淤青**: 包扎即可，无需手术
- **流血伤口**: 必须尽快包扎止血，流血速度决定优先级
- **感染**: 需要抗生素 + 卧床休息
- **致命伤**: 立即用最好的医药治疗
- **永久伤害（疤痕/旧伤）**: 需手术更换仿生部件

## 手术准备
- 手术前确保患者已麻醉（大多数手术自动麻醉）
- 手术室应干净无菌（降低感染率）
- 医生技能越高成功率越高
- 使用闪耀世界医药可大幅提升成功率
- 生命体征监测仪放在医疗床旁（+成功率）

## 植入体选择策略
| 植入体 | 适用殖民者 | 优先级 |
|--------|-----------|--------|
| 仿生腿 | 搬运工、矿工 | ★★★ |
| 仿生手臂 | 手工、锻造、建造工 | ★★★ |
| 仿生眼 | 射击手、研究者 | ★★ |
| 仿生心脏 | 全体，提升耐力 | ★★ |
| 疼痛阻断器 | 战斗员 | ★ |
| 肾上腺素肋 | 战斗员 | ★★ |
| 学习助手 | 研究者 | ★★ |

## 异种基因植入
- 异种基因胚需从基因包组装
- 植入后永久改变殖民者基因
- 基因复杂度受殖民者代谢限制
- 优先选择体质增强类基因（+移动/+操作）
- 避免冲突基因标签

## 药物使用
- **盘诺西林**: 预防感染，每 5 天服用一次
- **清醒药**: 临时提升工作效率，注意成瘾风险
- **精神茶**: 提升心情，成瘾风险较低
- **魔鬼素**: 战斗爆发，但成瘾后果严重，谨慎使用
