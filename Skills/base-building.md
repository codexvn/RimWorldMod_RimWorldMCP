# 基地建造规划指南

## 基地模板（强制优先使用）

建造多房间基地时，**必须优先使用 apply_base_template**，严禁自行计算坐标。LLM 的空间数学不可靠，手工算坐标必然导致房间错位。

### 工作流
1. `list_base_templates` — 查看可用模板及描述
2. 根据殖民地需求和地形选择合适的模板
3. `apply_base_template(template_name, center_x, center_y)` — 获取所有房间精确坐标
4. `designate_room` — 按返回坐标逐一建造
5. 根据实际需要微调（如跳过某个房间、调整单间尺寸）

### 模板选择
- 初期/独立建筑 → `single_room`（单房间，内径 5~13）
- 中期综合基地 → `nine_grid`（九宫格 3×3，紧凑无走廊）或 `nine_grid_walled`（带外围防御墙）
- 殖民者居住区 → `bedroom_row`（卧室排，5间×5x5 内径）

### 禁止事项
- **严禁**自行计算坐标后调 `designate_room`（房间必然错位）
- **严禁**用 `designate_build` 逐格造墙（效率低且更易出错）
- **严禁**在无 `get_structure_layout` 确认空地的情况下直接建造

## 标准房间尺寸
- **13x13 (内径)**: 最常见的万能尺寸，可容纳任意工作台配置，被 4 根柱子完美支撑
- **11x11**: 紧凑型房间，适合卧室、研究室
- **5x5 或 5x6**: 标准卧室尺寸
- **7x7**: 小型工作间（单一工作台 + 存储）

## 柱子与屋顶支撑
- 无支撑屋顶最大跨度 6 格
- 13x13 房间在四角向内 1 格各放一根柱子即可支撑整间
- 大型空间需要额外的柱子或墙壁

## 功能分区建议
1. **生产区**: 锻造台 + 裁缝台 + 机械加工台（相邻放置，方便共享存储）
2. **厨房区**: 炉灶 + 屠宰台，靠近冷库（减少食材搬运距离）
3. **研究区**: 研究台 + 精密装配台（需要无菌地板提升研究速度）
4. **医疗区**: 医疗床 + 生命体征监测仪，靠近药品库存
5. **居住区**: 卧室集群，每间 5x5，相邻放置减少走路时间
6. **存储区**: 冷库靠近厨房，普通仓库靠近生产区

## 材料选择
- **墙体**: 花岗岩 > 石灰岩 > 砂岩 > 大理石（耐久度排序）。大理石美观度高适合内部墙。
- **地板**: 无菌地板用于研究和医疗，钢地板用于生产区，木地板用于居住区
- **防御墙**: 花岗岩最佳（最高耐久），至少 2 格厚度

## 走廊设计
- 主走廊 3 格宽（避免拥堵）
- 室外通道 1 格即可，连接各建筑
- 重要走廊两端设门（防虫灾和空降袭击）

## 电力规划
- 电线埋在墙内美观且防火
- 关键设施用双路供电（主电网 + 电池备份）
- 电池室独立建造（防火隔离）
- 地热发电机用围墙保护

## 温度控制
- 每间房装通风口连接到走廊（走廊统一控温）
- 冷库双层墙壁 + 双层门（气闸设计）
- 加热器/空调各 1-2 台即可控制标准房间温度

## 防御布局
- 外墙至少 2 格厚度（花岗岩）
- 拐角处设置射击位（沙袋 + 屋顶掩体）
- 入口设多个（方便突围，但需设陷阱走廊引导敌人）
- 重要设施（发电、库存）放在基地内部
